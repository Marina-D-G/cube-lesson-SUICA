# cube-lesson-SUICA

Learning Project – HTML, CSS, JavaScript + suica and lil-gui

This project was created for learning purposes and demonstrates core frontend skills by building a modern web interface using HTML, CSS, and JavaScript, along with the Suica ana lil-gui libraries. It presents a student lesson for the shortest path in a cube and has an interactive board.

How to use - Clone the repository:

- git clone https://github.com/your-username/project-name.git
- Open the project folder.
- Run index.html in your browser.

💡 Note
This is a learning project and is not intended for production use.
